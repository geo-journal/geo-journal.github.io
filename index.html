Scientific Computing Journal Club


A preprint/journal club centered around topics of data science, scientific computing, and geoscience applications of these topics.

How It Works

  1. Keep the commitment light - one paper per month to accommodate everyone’s schedules
  2. We rotate through the group members (alphabetically by surname for simplicity) with each month having a new discussion leader
  3. Each month the discussion leader is tasked with selecting that month’s paper. The group member is also asked to provide a few discussion questions (detailed below).
  4. We use a github repository to host this site as well as the paper discussions
  5. Whenever possible we will store PDFs of the papers we review. Papers that are not open access will be distributed to the group via email
  6. Discussion is asynchronous, public, and via GitHub issues.

Discussion Leader's Role

  1. Select a paper before the first day of the month you are to lead
  2. On the first day of the month distribute your paper to the group - either via email or, for open access papers, by committing a PDF to the "papers" directory in the GitHub repository
  3. Create a set of 5 to 10 discussion questions to stimulate the initial discussion
  4. Create a new GitHub Issue with your discussion questions and link to the paper (if applicable)
  5. Discussion proceeds via commenting on the GitHub Issue
  6. Members may want to set their GitHub settings to receive email notifications of new comments and mentions
